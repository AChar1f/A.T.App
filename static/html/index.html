Users
HTTP Method Path Description
GET /users Fetches all Users data from the database
GET /users/:id Fetches a single User's data from the database for the id provided
GET /admin Gets all admin users
POST /admin/register Adds a new admin user to the database.
PATCH /admin/:id/update Updates a admin user for the id value provided.
